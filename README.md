# EcoWave: "작은 실천이 만드는 큰 파도"

> AI 기반 브랜드 광고 캠페인 스토리보드

---

## 1. 브랜드 아이덴티티 및 캠페인 개요

| 항목 | 내용 |
|---|---|
| **브랜드 아이덴티티** | 지속 가능한 미래를 위해 세련된 친환경 제품을 제안하는 라이프스타일 브랜드 |
| **캠페인 목표** | 일회용품 사용 줄이기 독려 및 EcoWave 텀블러 제품 인지도 확산 |
| **타겟** | 연령 및 성별 무관 |
| **USP** | 친환경 + 텀블러로 일상에서 실천 가능한 지속 가능한 미래 |
| **핵심 메시지** | "당신의 작은 선택이 지구를 살리는 큰 파도가 됩니다." |

## 5. 제약 사항 준수 내역

### ✅ 소스 및 저작권 (100% AI 생성)
- 직접 촬영하거나 유료 스톡(Shutterstock 등)을 전혀 사용하지 않음
- 모든 시각·청각 요소는 생성형 AI 결과물로만 구성
- 환경 보호라는 공익적 주제로 윤리적 이슈가 없는 클린한 콘텐츠

### ✅ 편집 도구 사용 범위 (통합 편집 용도로 제한)
- CapCut 등 타 도구는 오직 **컷 편집, 자막 삽입, 로고 합성**의 최소 용도로만 사용
- 핵심 비주얼(플라스틱이 쌓이는 장면, 텀블러의 질감)은 AI 생성물을 그대로 유지

### ✅ 비용 및 크레딧 효율화 전략
- **워크플로우**: 이미지 생성 → 비디오 변환 순서로 크레딧 낭비 방지
- 전체 씬을 3개로 압축하여 메시지 밀도를 높임
- 정지 이미지에 짧은 움직임만 부여해 영상 생성 크레딧 절약

### ✅ 도구 접근성 대응 (대체 도구 확보)
| 용도 | 기본 도구 | 대체 도구 |
|---|---|---|
| 이미지 | `gpt-image-2` | `gemini` |
| 비디오 | `sora-2` | `google veo` |

### ✅ 일관성(스타일) 유지 기술
| 파라미터 | 설정 | 목적 |
|---|---|---|
| Style Reference | `--sref [Scene 02 이미지 URL]` | 민트색 톤과 깨끗한 조명 스타일 고정 |
| Aspect Ratio | `--ar 9:16` | 숏폼 규격 통일 |

## 7. 전체 제작 파이프라인

```mermaid
graph LR
    A[기획 단계<br/>브랜드 정의 + 3씬 스토리보드] --> B[에셋 생성 단계<br/>gpt-image-2 → sora-2]
    B --> C[통합 및 보정 단계<br/>합성 · 색보정 · 자막]
    C --> D[최종 검수<br/>파일 네이밍 · 아카이빙 · 출력]
```

1. **기획 단계 (Planning)**: 브랜드 아이덴티티 정의 및 10초 분량 3씬 스토리보드 작성
2. **에셋 생성 단계 (Generation)**: `gpt-image-2`로 고품질 스틸 이미지 생성 → `sora-2`로 모션 부여
3. **통합 및 보정 단계 (Integration)**: 영상/오디오 합성, 색감 보정(Grading), 자막 삽입
4. **최종 검수 (Review)**: 파일 네이밍 규칙에 따른 아카이빙 및 최종 출력
